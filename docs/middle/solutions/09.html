🧮 중등부 9번: 인버전의 개수 (10점)

📘 문제 요약

1부터 8까지의 수가 한 번씩 등장하는 수열 \(a_1, a_2, \dots, a_8\)이 있고,
각 \(c_i\)는 자기 앞에 있으면서 자신보다 큰 수의 개수를 나타낸다.
다음과 같은 인버전 수 \(c\)가 주어졌을 때, 수열 \(a\)를 구하시오.

i12345678
ci01030523

🧠 풀이 전략

\(c_i\)는 “자기 앞에 있는 수 중 자기보다 큰 수의 개수”이다.
이 정의를 이용해, 오른쪽부터 왼쪽으로 수를 하나씩 채워나가며 유일한 수열을 구성할 수 있다.

1️⃣ a₈ = 5
남은 후보: [1, 2, 3, 4, 5, 6, 7, 8]
\(c_8 = 3\) → a₈ 앞에 자신보다 큰 수가 3개 있어야 한다.
→ 남은 수 중 자신보다 큰 수가 3개인 수는 5
2️⃣ a₇ = 6
남은 후보: [1, 2, 3, 4, 6, 7, 8]
\(c_7 = 2\) → 자신보다 큰 수가 앞에 2개 있어야 함
→ 남은 수 중 6이 조건을 만족
3️⃣ a₆ = 1
남은 후보: [1, 2, 3, 4, 7, 8]
\(c_6 = 5\) → 앞에 있는 모든 수가 자신보다 커야 함 → a₆는 가장 작은 수 1
4️⃣ a₅ = 8
남은 후보: [2, 3, 4, 7, 8]
\(c_5 = 0\) → 앞에 자신보다 큰 수가 없어야 함 → 가장 큰 수 8
5️⃣ a₄ = 2
남은 후보: [2, 3, 4, 7]
\(c_4 = 3\) → 앞에 자신보다 큰 수 3개 있어야 함 → 2가 적절
6️⃣ a₃ = 7
남은 후보: [3, 4, 7]
\(c_3 = 0\) → 앞에 자신보다 큰 수 없음 → 가장 큰 수 7
7️⃣ a₂ = 3
남은 후보: [3, 4]
\(c_2 = 1\) → 앞에 자신보다 큰 수 1개 필요 → 3이 적절
8️⃣ a₁ = 4
남은 후보: [4]
마지막 남은 수 = 4

✅ 최종 수열
