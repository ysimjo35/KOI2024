🕵️‍♂️ 중등부 8번: 기사, 도둑, 바보 (8점)

📘 문제 요약

세 사람 A, B, C가 있습니다. 각각의 사람은 기사(항상 진실), 도둑(항상 거짓), 바보(진실 또는 거짓) 중 하나입니다. 역할은 서로 다릅니다.

세 사람의 발언은 다음과 같습니다:

  • A: 우리 중 한 명 이상이 바보이다.
  • B: A는 바보가 아니다.
  • C: 나는 도둑이다.

🧠 논리적 추론으로 좁혀가기

1️⃣ C의 말: "나는 도둑이다"
이 말이 참이면 → C는 도둑이어야 함 → 도둑이 참말을 한 것 → ❌ 모순 발생
이 말이 거짓이면 → 실제로 C는 도둑이 아님 → OK

그런데 만약 C가 기사라면, 그 말은 참이어야 하므로 역시 ❌ 모순
✅ 따라서 C는 반드시 바보입니다.
2️⃣ A의 말: "우리 중 한 명 이상이 바보이다"
현재 C가 바보로 확정되었으므로 이 말은 참입니다.
이 말이 참이 되기 위해서는 A는 기사나 바보여야 합니다.
(도둑은 거짓말만 해야 하므로 참말을 할 수 없음) ✅ 따라서 A는 기사 또는 바보입니다.
3️⃣ B의 말: "A는 바보가 아니다"
이 말이 참이면 → A는 기사 또는 도둑
이 말이 거짓이면 → A는 바보여야 함
→ 따라서 B의 말이 참인지 거짓인지에 따라 A의 역할도 결정됨

이제 B가 기사/도둑/바보인 경우를 나눠보며 가능한 조합을 따져보자.

📋 가능한 조합 정리 (C = 바보 고정)

  • Case 1: A = 기사, B = 도둑 → A의 말: 참 ✅ / B의 말: 거짓 → A는 바보여야 하지만 기사 ❌ → 모순
  • Case 2: A = 기사, B = 기사 → A의 말: 참 ✅ / B의 말: 참 → A는 기사 → ✅ 가능
  • Case 3: A = 기사, B = 바보 → A는 참 / B는 말 상관없음 → ✅ 가능
  • Case 4: A = 바보, B = 도둑 → A 말은 참 / 도둑인 B는 “A는 바보 아니다”라고 말했는데 → 거짓이어야 함 → 실제로 A는 바보 → ✅ 가능
  • Case 5: A = 바보, B = 기사 → A 말은 참 / 기사 B가 “A는 바보 아니다” → 참이 되어야 하는데 실제 A는 바보 → ❌ 모순
  • Case 6: A = 바보, B = 바보 → 바보는 뭐든 말할 수 있음 → ✅ 가능
